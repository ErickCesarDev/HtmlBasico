Tag <h1> até <h6>

A tag <h1> até <h6> serve para definir titulos, sendo que quanto menor o numero maior a importancia do titulo.

Sintaxe

<h1> texto </h1>
<h2> texto </h2>
<h3> texto </h3>
<h4> texto </h4>
<h5> texto </h5>
<h6> texto </h6>
